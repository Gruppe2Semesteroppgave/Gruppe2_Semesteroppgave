Om prosjektet

Kort om siden

Merk: Denne siden beskriver studentprosjektet (DAT111/ING100) – leveranser, metode og milepæler. Info om det fiktive selskapet Regnbyen Bergen A/S ligger på Hjem.

Visjon

Å gjøre hverdagen i Bergen enklere og mer bærekraftig i regnvær gjennom et brukervennlig nettsted med praktiske råd og en enkel ML-basert nedbør­estimering.

Målsetting

  • Nettsted: Klar, tilgjengelig info om klær, bolig, transport.
  • Dokumentasjon: Rapport iht. ING100 (backlog, Gantt, wireframes).
  • Teknologi: Enkel Python-estimering (Innlevering 3).
  • Samarbeid: Scrum-inspirert arbeidsflyt og roller.

Leveranser

  • Nettsted (HTML → senere CSS)
  • Prosjektrapport
  • Python-modul for nedbør

Metodikk

  • 3 sprint med milepæler
  • Product backlog (prioritet + estimater)
  • Wireframes før implementasjon
  • W3C-validering

Milepæler

  • Innlevering 1 (uke 38): HTML-grunnstruktur
  • Innlevering 2 (uke 41): Felles CSS + innhold
  • Innlevering 3 (uke 44): Python + sluttrapport
